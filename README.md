🏛 حزب العقوق

موقع ساخر تم تطويره باستخدام HTML, CSS, JavaScript بهدف الترفيه وتجربة بناء موقع Front-End بسيط.

✨ المميزات

- 🪪 إصدار بطاقة عضوية.
- 🚗 لوحة حزب العقوق.
- 📊 اختبار نسبة العقوق.
- 🎡 عجلة الحظ.
- 🏆 نظام الرتب والأوسمة.
- 📜 دستور الحزب.
- 📈 إحصائيات الحزب.
- 📱 تصميم متوافق مع الهواتف.

🛠️ التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript (Vanilla)
